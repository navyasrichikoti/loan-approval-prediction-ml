# Loan Approval Prediction

## Overview

Loan Approval Prediction is a Machine Learning based web application that predicts whether a loan application is likely to be approved based on applicant details such as income, credit history, education, employment status, loan amount, and property area.

The application is built using Flask, Python, HTML, CSS, Bootstrap, and Machine Learning algorithms to provide quick and accurate loan eligibility predictions.

---

## Features

* Loan Approval Prediction using Machine Learning
* User-Friendly and Responsive Interface
* Real-Time Prediction Results
* Clean and Professional UI
* Input Validation and Error Handling

## Installation

Clone the repository:

```bash
git clone https://github.com/navyasrichikoti/loan-approval-prediction-ml.git
```

Navigate to project folder:

```bash
cd Loan-Approval-Prediction-main
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
